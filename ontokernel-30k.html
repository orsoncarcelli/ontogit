onto-research

Autonomous ML research lab for ontology-enriched prediction market forecasting.

Core question: Does a knowledge graph (ontology) of market entities, built by pred-markets, improve prediction accuracy over tabular features alone?

How it works

This repo adapts the autoresearch pattern: an AI agent edits train.py, runs 5-minute experiments, and keeps or discards changes based on whether Brier score improved. The domain is binary prediction market forecasting using a three-tier feature system:

Tier Source Features
0 Market metadata implied_prob, volume, liquidity, days_to_end, category, bag-of-words
1 Filtered ontology graph degree centrality, predicate-typed neighbor counts, confidence stats
2 Filtered graph embeddings spectral embeddings of entity neighborhoods (dim=16)

The agent toggles FEATURE_TIERS = [0, 1, 2] in the CONFIG block to ablate which tiers are active, directly measuring the value of ontology-derived features. Tier 1/2 are built from a leakage-safe feature graph filtered to the training cutoff timestamp and excluding nodes tagged with resolution / ml_training.

Quick start

Requirements: Python 3.11+, uv, NVIDIA GPU (optional, CPU works).

# 1. Install dependencies
uv sync --locked

# 2. Export the reproducible data snapshot from pred-markets
#    This writes data/ontology.json, data/resolved.csv, and data/snapshot.json
uv run python scripts/export_data.py --pred-markets-dir ../prediction-markets

# 3. Verify the snapshot, leakage audit, and baseline tables
uv run prepare.py

# 4. Run a single experiment (~5 min total, shared across seeds)
uv run train.py

# 5. Launch autonomous agent
#    Point your agent at program.md and let it iterate on train.py

If train.py omits snapshot_hash, the snapshot is missing or stale. Re-run uv run python scripts/export_data.py --pred-markets-dir ../prediction-markets. If the exporter reports that prediction-markets has no resolved_markets table yet, seed it there first with uv run python scripts/refresh_markets.py --include-resolved.

Relationship to other repos

  • ontokernel — the ontology kernel (typed knowledge graph engine). onto-research uses it for typed graph access and schema validation.
  • pred-markets — the primary data source. Builds the ontology from Polymarket data, stores resolved markets for training labels. onto-research consumes its data via flat-file exports (no runtime dependency).

Design choices

  • Flat-file data snapshot: ontology.json, resolved.csv, and snapshot.json are exported from pred-markets, not imported at runtime. The snapshot file stores file hashes so prepare.py / train.py can verify provenance.
  • Leakage-safe feature graph: Tier 1/2 features are computed from a graph filtered to the last training market's closed_time, with resolution / ml_training nodes excluded. This keeps post-resolution ontology artifacts out of validation features.
  • Current data caveat: with the current exported snapshot, the leakage-safe cutoff removes all ontology edges. That means Tier 1/2 collapse to zero-valued features today; this is an honest result, not a runtime bug.
  • Three-tier ablation: the feature system is designed for controlled experiments. Compare tier 0 vs 0+1 vs 0+1+2 to isolate the value of graph structure when temporally valid graph signal exists.
  • Fixed time budget: training always runs for exactly 5 minutes, shared across the configured seeds. This makes experiments directly comparable regardless of what the agent changes.
  • Single file to modify: the agent only touches train.py. Keeps diffs reviewable and experiments focused.

"""Data loading, feature extraction, and evaluation for ontology-ML experiments.

THIS FILE IS READ-ONLY — the autoresearch agent must NOT modify it.
It provides the fixed evaluation harness, data loading, and feature extraction
that train.py consumes.

Three-tier feature system:
  Tier 0 — Tabular baseline (market metadata only)
  Tier 1 — Graph-structural (degree, predicate counts, confidence stats)
  Tier 2 — Graph embeddings (spectral embeddings of entity neighborhoods)
"""
from __future__ import annotations

import csv
import hashlib
import json
import math
import re
from collections import Counter
from pathlib import Path
from typing import Any

import networkx as nx
import numpy as np

# ── Constants ────────────────────────────────────────────────────────────

DATA_DIR = Path(__file__).resolve().parent / "data"
ONTOLOGY_PATH = DATA_DIR / "ontology.json"
MARKETS_PATH = DATA_DIR / "resolved.csv"
TIME_BUDGET = 300  # 5 minutes wall-clock for training
EMBEDDING_DIM = 16  # spectral embedding dimensionality

# ── Ontology loading ─────────────────────────────────────────────────────


def load_ontology_graph(path: Path = ONTOLOGY_PATH) -> nx.DiGraph:
    """Load ontology from NetworkX node-link JSON into a DiGraph.

    Falls back gracefully: returns an empty graph if the file is missing,
    so tier-0-only experiments can still run without ontology data.
    """
    if not path.exists():
        return nx.DiGraph()
    raw = json.loads(path.read_text(encoding="utf-8"))
    try:
        G = nx.node_link_graph(raw, directed=True)
    except Exception:
        G = nx.DiGraph()
        for node in raw.get("nodes", []):
            G.add_node(node.get("id", str(node)))
        for link in raw.get("links", []):
            G.add_edge(
                link["source"],
                link["target"],
                predicate=link.get("predicate", "related_to"),
                confidence=float(link.get("confidence", 0.7)),
                timestamp=float(link.get("timestamp", 0.0)),
            )
    return G


def load_ontology_kernel(path: Path = ONTOLOGY_PATH) -> Any:
    """Load ontology via the ontokernel typed API (optional, for typed access).

    Returns an OntologyGraph instance backed by the kernel, or None if
    the kernel is not installed or the file is missing.
    """
    if not path.exists():
        return None
    try:
        from ontokernel import OntologyGraph
        from ontokernel.config import KernelConfig

        config = KernelConfig(
            backend="networkx",
            persist_path=str(path),
            default_namespace="polymarket",
        )
        graph = OntologyGraph(config=config)
        graph.load()
        return graph
    except Exception:
        return None


# ── Market data loading ──────────────────────────────────────────────────


def load_markets(path: Path = MARKETS_PATH) -> list[dict]:
    """Load resolved markets from CSV, sorted by closed_time (oldest first)."""
    if not path.exists():
        raise FileNotFoundError(
            f"No market data at {path}. Run: uv run python scripts/export_data.py"
        )
    with open(path, encoding="utf-8") as f:
        reader = csv.DictReader(f)
        rows = list(reader)
    for r in rows:
        for col in ("volume", "liquidity", "implied_prob_at_close"):
            r[col] = _safe_float(r.get(col), 0.0)
        r["resolved_yes"] = int(float(r.get("resolved_yes", 0)))
        r["tags"] = _parse_json(r.get("tags", "[]"), [])
    rows.sort(key=lambda r: r.get("closed_time", ""))
    return rows


def time_split(rows: list[dict], train_frac: float = 0.7) -> tuple[list[dict], list[dict]]:
    """Temporal split — older rows for train, newer for validation."""
    if not rows:
        return [], []
    n = max(1, int(len(rows) * train_frac))
    return rows[:n], rows[n:]


# ── Evaluation ───────────────────────────────────────────────────────────


def evaluate_brier(y_true: np.ndarray, y_prob: np.ndarray) -> float:
    """Brier score: mean squared error between predicted probabilities and labels.

    Lower is better. Range [0, 1]. A naive 0.5 baseline scores 0.25.
    """
    y_true = np.asarray(y_true, dtype=np.float64)
    y_prob = np.asarray(y_prob, dtype=np.float64).clip(0.0, 1.0)
    return float(np.mean((y_prob - y_true) ** 2))


def evaluate_log_loss(
    y_true: np.ndarray, y_prob: np.ndarray, eps: float = 1e-15
) -> float:
    """Binary cross-entropy / log-loss.  Lower is better."""
    y_prob = np.clip(np.asarray(y_prob, dtype=np.float64), eps, 1 - eps)
    y_true = np.asarray(y_true, dtype=np.float64)
    return float(-np.mean(y_true * np.log(y_prob) + (1 - y_true) * np.log(1 - y_prob)))


def evaluate_calibration_error(
    y_true: np.ndarray, y_prob: np.ndarray, n_bins: int = 10
) -> float:
    """Expected calibration error (ECE).

    Weighted average of |mean_predicted - mean_actual| across bins.
    Returns 0.0 for perfect calibration.
    """
    y_prob = np.asarray(y_prob, dtype=np.float64)
    y_true = np.asarray(y_true, dtype=np.float64)
    n = len(y_prob)
    if n == 0:
        return 0.0
    bins = np.linspace(0.0, 1.0, n_bins + 1)
    ece = 0.0
    for lo, hi in zip(bins[:-1], bins[1:]):
        mask = (y_prob >= lo) & (y_prob <= hi) if hi >= 1.0 else (y_prob >= lo) & (y_prob < hi)
        count = int(mask.sum())
        if count > 0:
            ece += count * abs(float(y_prob[mask].mean()) - float(y_true[mask].mean()))
    return round(ece / n, 6)


def evaluate_auc(y_true: np.ndarray, y_prob: np.ndarray) -> float:
    """ROC AUC score, returns 0.5 when only one class is present."""
    y_true = np.asarray(y_true, dtype=np.float64)
    y_prob = np.asarray(y_prob, dtype=np.float64)
    if len(np.unique(y_true)) < 2:
        return 0.5
    from sklearn.metrics import roc_auc_score
    return float(roc_auc_score(y_true, y_prob))


def evaluate_all(y_true: np.ndarray, y_prob: np.ndarray) -> dict[str, float]:
    """Full metric suite — returns dict with brier, log_loss, calibration_error, auc."""
    return {
        "brier": evaluate_brier(y_true, y_prob),
        "log_loss": evaluate_log_loss(y_true, y_prob),
        "calibration_error": evaluate_calibration_error(y_true, y_prob),
        "auc": evaluate_auc(y_true, y_prob),
    }


# ── Baselines ────────────────────────────────────────────────────────────


def baseline_constant(
    y_train: np.ndarray, y_val: np.ndarray
) -> tuple[np.ndarray, float]:
    """Predict the training-set mean probability for every validation sample."""
    mean_p = float(np.mean(y_train))
    preds = np.full(len(y_val), mean_p)
    return preds, evaluate_brier(y_val, preds)


def baseline_market_implied(
    val_rows: list[dict], y_val: np.ndarray
) -> tuple[np.ndarray, float]:
    """Use implied_prob_at_close directly as the prediction."""
    preds = np.array(
        [_safe_float(r.get("implied_prob_at_close", 0.5), 0.5) for r in val_rows],
        dtype=np.float64,
    )
    return preds, evaluate_brier(y_val, preds)


def baseline_logistic(
    X_train: np.ndarray, y_train: np.ndarray,
    X_val: np.ndarray, y_val: np.ndarray,
) -> tuple[np.ndarray, float]:
    """LogisticRegression baseline (sklearn, max_iter=1000)."""
    from sklearn.linear_model import LogisticRegression
    clf = LogisticRegression(max_iter=1000, solver="lbfgs")
    clf.fit(X_train, y_train)
    preds = clf.predict_proba(X_val)[:, 1]
    return preds, evaluate_brier(y_val, preds)


def baseline_tree(
    X_train: np.ndarray, y_train: np.ndarray,
    X_val: np.ndarray, y_val: np.ndarray,
) -> tuple[np.ndarray, float]:
    """HistGradientBoostingClassifier baseline (max_iter=100, max_depth=3)."""
    from sklearn.ensemble import HistGradientBoostingClassifier
    clf = HistGradientBoostingClassifier(max_iter=100, max_depth=3, random_state=42)
    clf.fit(X_train, y_train)
    preds = clf.predict_proba(X_val)[:, 1]
    return preds, evaluate_brier(y_val, preds)


def run_all_baselines(
    X_train: np.ndarray, y_train: np.ndarray,
    X_val: np.ndarray, y_val: np.ndarray,
    val_rows: list[dict],
) -> dict[str, dict[str, Any]]:
    """Run all baselines and return {name: {"brier": float, "preds": ndarray}}."""
    results: dict[str, dict[str, Any]] = {}

    preds, brier = baseline_constant(y_train, y_val)
    results["constant"] = {"brier": brier, "preds": preds}

    preds, brier = baseline_market_implied(val_rows, y_val)
    results["market_implied"] = {"brier": brier, "preds": preds}

    preds, brier = baseline_logistic(X_train, y_train, X_val, y_val)
    results["logistic"] = {"brier": brier, "preds": preds}

    preds, brier = baseline_tree(X_train, y_train, X_val, y_val)
    results["tree"] = {"brier": brier, "preds": preds}

    return results


# ── Leakage detection ────────────────────────────────────────────────────


def leakage_report(
    G: nx.DiGraph, cutoff_timestamp: float,
    exclude_sources: set[str] | None = None,
) -> dict[str, Any]:
    """Audit how many nodes/edges survive temporal filtering vs are removed.

    Uses raw NetworkX graph for reporting. The kernel's temporal query params
    are the runtime mechanism; this function is for human-readable diagnostics.
    """
    if exclude_sources is None:
        exclude_sources = {"resolution", "ml_training"}

    total_nodes = G.number_of_nodes()
    total_edges = G.number_of_edges()

    removed_edges = 0
    for _, _, data in G.edges(data=True):
        ts = float(data.get("timestamp", 0.0))
        src = str(data.get("source", ""))
        if ts > cutoff_timestamp or src in exclude_sources:
            removed_edges += 1

    removed_nodes = 0
    for node in G.nodes():
        ndata = G.nodes[node]
        fs = float(ndata.get("first_seen", 0.0))
        sources = ndata.get("sources", [])
        if isinstance(sources, str):
            sources = _parse_json(sources, [])
        if fs > cutoff_timestamp:
            removed_nodes += 1
        elif any(s in exclude_sources for s in (sources or [])):
            removed_nodes += 1

    return {
        "total_nodes": total_nodes,
        "total_edges": total_edges,
        "surviving_nodes": total_nodes - removed_nodes,
        "surviving_edges": total_edges - removed_edges,
        "removed_nodes": removed_nodes,
        "removed_edges": removed_edges,
        "cutoff_timestamp": cutoff_timestamp,
        "exclude_sources": sorted(exclude_sources),
    }

def _sha256_file(path: Path) -> str | None:
    """Return the SHA-256 hex digest for a file, or None if it is missing."""
    if not path.exists():
        return None
    h = hashlib.sha256()
    with open(path, "rb") as f:
        for chunk in iter(lambda: f.read(1 << 16), b""):
            h.update(chunk)
    return h.hexdigest()


def load_snapshot(path: Path = SNAPSHOT_PATH) -> dict[str, Any] | None:
    """Load snapshot.json if it exists, else None."""
    if not path.exists():
        return None
    return json.loads(path.read_text(encoding="utf-8"))


def snapshot_status(path: Path = SNAPSHOT_PATH) -> tuple[str, str | None]:
    """Return (status, snapshot_hash) for snapshot.json.

    Status is one of: ``missing``, ``invalid``, ``stale``, or ``valid``.
    """
    snap = load_snapshot(path)
    if snap is None:
        return "missing", None

    snap_hash = snap.get("snapshot_hash")
    csv_hash = snap.get("resolved_csv_sha256")
    ont_hash = snap.get("ontology_json_sha256")
    if not isinstance(snap_hash, str) or not snap_hash:
        return "invalid", None
    if not isinstance(csv_hash, str) or not isinstance(ont_hash, str):
        return "invalid", None

    current_csv_hash = _sha256_file(MARKETS_PATH)
    current_ont_hash = _sha256_file(ONTOLOGY_PATH)
    if current_csv_hash is None or current_ont_hash is None:
        return "stale", None
    if current_csv_hash != csv_hash or current_ont_hash != ont_hash:
        return "stale", None

    return "valid", snap_hash


def verify_snapshot(path: Path = SNAPSHOT_PATH) -> str | None:
    """Return the snapshot_hash if snapshot.json exists and is valid, else None."""
    status, snap_hash = snapshot_status(path)
    return snap_hash if status == "valid" else None


def build_feature_graph(
    G: nx.DiGraph,
    cutoff_timestamp: float | None = None,
    exclude_sources: set[str] | frozenset[str] | None = None,
) -> nx.DiGraph:
    """Return a leakage-safe graph view for feature extraction.

    Nodes added after the training cutoff, or tagged with excluded sources,
    are removed. Remaining edges are filtered by timestamp/source as well.
    """
    if G.number_of_nodes() == 0:
        return nx.DiGraph()

    excluded = {str(s) for s in (exclude_sources or set())}
    if cutoff_timestamp is None and not excluded:
        return G.copy()

    filtered = nx.DiGraph()
    filtered.graph.update(G.graph)

    for node, data in G.nodes(data=True):
        first_seen = float(data.get("first_seen", 0.0))
        sources = data.get("sources", [])
        if isinstance(sources, str):
            sources = _parse_json(sources, [])
        sources = [str(s) for s in (sources or [])]

        if cutoff_timestamp is not None and first_seen > cutoff_timestamp:
            continue
        if excluded and any(src in excluded for src in sources):
            continue

        filtered.add_node(node, **data)

    for src, dst, data in G.edges(data=True):
        if src not in filtered or dst not in filtered:
            continue
        ts = float(data.get("timestamp", 0.0))
        source = str(data.get("source", ""))
        if cutoff_timestamp is not None and ts > cutoff_timestamp:
            continue
        if excluded and source in excluded:
            continue
        filtered.add_edge(src, dst, **data)

    return filtered


# ── Tier 0: Tabular features ────────────────────────────────────────────


def build_category_map(rows: list[dict]) -> dict[str, int]:
    """Deterministic category -> integer encoding from training rows."""
    cats = sorted({r.get("category", "") or "" for r in rows})
    return {c: i for i, c in enumerate(cats)}


def build_vocab(rows: list[dict], max_vocab: int = 200) -> list[str]:
    """Small bag-of-words vocabulary from market questions."""
    counter: Counter[str] = Counter()
    for r in rows:
        counter.update(_tokenize(r.get("question", "")))
    return [w for w, _ in counter.most_common(max_vocab)]


def extract_tier0(
    row: dict,
    category_map: dict[str, int] | None = None,
    vocab: list[str] | None = None,
) -> dict[str, float]:
    """Tabular features from market metadata only — no graph needed."""
    implied = _safe_float(row.get("implied_prob_at_close", 0.5), 0.5)
    volume = _safe_float(row.get("volume"), 0.0)
    liquidity = _safe_float(row.get("liquidity"), 0.0)
    tags = row.get("tags", [])
    tag_count = len(tags) if isinstance(tags, list) else 0
    days = _days_to_end(row)

    feats: dict[str, float] = {
        "implied_prob": implied,
        "log_volume": math.log1p(volume),
        "log_liquidity": math.log1p(liquidity),
        "days_to_end": days,
        "tag_count": float(tag_count),
    }

    cat = row.get("category", "") or ""
    feats["category_enc"] = float(category_map.get(cat, -1)) if category_map else 0.0

    if vocab:
        tokens = set(_tokenize(row.get("question", "")))
        for w in vocab:
            feats[f"bow_{w}"] = 1.0 if w in tokens else 0.0

    return feats


# ── Tier 1: Graph-structural features ───────────────────────────────────


def extract_tier1(
    row: dict,
    G: nx.DiGraph,
    predicate_list: list[str] | None = None,
) -> dict[str, float]:
    """Graph-structural features aggregated over a market's mentioned entities.

    For each entity mentioned in the market question, computes degree,
    predicate-typed neighbor counts, and confidence statistics, then
    aggregates (mean) across all matched entities.
    """
    if G.number_of_nodes() == 0:
        return _empty_tier1(predicate_list)

    entities = _match_entities(row, G)
    if not entities:
        return _empty_tier1(predicate_list)

    if predicate_list is None:
        predicate_list = _get_predicate_list(G)

    accum: dict[str, list[float]] = {}

    for ent in entities:
        f = _entity_graph_features(ent, G, predicate_list)
        for k, v in f.items():
            accum.setdefault(k, []).append(v)

    return {k: float(np.mean(vs)) for k, vs in accum.items()}


def _entity_graph_features(
    entity: str,
    G: nx.DiGraph,
    predicate_list: list[str],
) -> dict[str, float]:
    """Graph features for a single entity node."""
    in_deg = G.in_degree(entity) if entity in G else 0
    out_deg = G.out_degree(entity) if entity in G else 0

    feats: dict[str, float] = {
        "g_in_degree": float(in_deg),
        "g_out_degree": float(out_deg),
        "g_total_degree": float(in_deg + out_deg),
    }

    confidences: list[float] = []
    pred_counts: dict[str, int] = {p: 0 for p in predicate_list}

    if entity in G:
        for _, _, data in G.out_edges(entity, data=True):
            c = float(data.get("confidence", 0.7))
            confidences.append(c)
            p = data.get("predicate", "related_to")
            if p in pred_counts:
                pred_counts[p] += 1
        for _, _, data in G.in_edges(entity, data=True):
            c = float(data.get("confidence", 0.7))
            confidences.append(c)
            p = data.get("predicate", "related_to")
            if p in pred_counts:
                pred_counts[p] += 1

    feats["g_mean_confidence"] = float(np.mean(confidences)) if confidences else 0.0
    feats["g_max_confidence"] = float(np.max(confidences)) if confidences else 0.0
    feats["g_min_confidence"] = float(np.min(confidences)) if confidences else 0.0

    for p in predicate_list:
        feats[f"g_pred_{p}"] = float(pred_counts.get(p, 0))

    return feats


def _empty_tier1(predicate_list: list[str] | None = None) -> dict[str, float]:
    """Zero-valued tier-1 features when no graph or no entity match."""
    preds = predicate_list or []
    feats: dict[str, float] = {
        "g_in_degree": 0.0,
        "g_out_degree": 0.0,
        "g_total_degree": 0.0,
        "g_mean_confidence": 0.0,
        "g_max_confidence": 0.0,
        "g_min_confidence": 0.0,
    }
    for p in preds:
        feats[f"g_pred_{p}"] = 0.0
    return feats


def _match_entities(row: dict, G: nx.DiGraph) -> list[str]:
    """Find graph nodes mentioned in a market's question or tags."""
    question = (row.get("question", "") or "").lower()
    tags = row.get("tags", [])
    if isinstance(tags, str):
        tags = _parse_json(tags, [])
    tag_strs = {str(t).lower() for t in (tags or [])}

    query_tokens = set(_tokenize(question))
    matched: list[str] = []

    for node in G.nodes():
        node_lower = str(node).lower()
        node_tokens = set(_tokenize(node_lower))
        if node_lower in tag_strs:
            matched.append(node)
        elif node_tokens and node_tokens.issubset(query_tokens):
            matched.append(node)

    return matched


def _get_predicate_list(G: nx.DiGraph) -> list[str]:
    """Extract sorted unique predicates from the graph."""
    preds: set[str] = set()
    for _, _, data in G.edges(data=True):
        p = data.get("predicate")
        if p:
            preds.add(str(p))
    return sorted(preds)


# ── Tier 2: Graph embeddings ────────────────────────────────────────────


def compute_spectral_embeddings(
    G: nx.DiGraph,
    dim: int = EMBEDDING_DIM,
) -> dict[str, np.ndarray]:
    """Spectral embeddings for each node in the graph.

    Uses the Laplacian eigenvectors of the undirected projection.
    Returns a dict mapping node ID -> embedding vector of shape (dim,).
    Falls back to zero vectors if the graph is too small.
    """
    if G.number_of_nodes() < dim + 1:
        return {n: np.zeros(dim) for n in G.nodes()}

    U = G.to_undirected()
    try:
        L = nx.laplacian_matrix(U).astype(np.float64)
        from scipy.sparse.linalg import eigsh
        _, vecs = eigsh(L, k=dim + 1, which="SM")
        vecs = vecs[:, 1:]  # skip first trivial eigenvector
    except Exception:
        return {n: np.zeros(dim) for n in G.nodes()}

    nodes = list(U.nodes())
    return {nodes[i]: vecs[i] for i in range(len(nodes))}


def extract_tier2(
    row: dict,
    G: nx.DiGraph,
    embeddings: dict[str, np.ndarray],
    dim: int = EMBEDDING_DIM,
) -> dict[str, float]:
    """Embedding features — mean of matched entity embeddings."""
    entities = _match_entities(row, G)
    if not entities:
        return {f"emb_{i}": 0.0 for i in range(dim)}

    vecs = [embeddings[e] for e in entities if e in embeddings]
    if not vecs:
        return {f"emb_{i}": 0.0 for i in range(dim)}

    mean_vec = np.mean(vecs, axis=0)
    return {f"emb_{i}": float(mean_vec[i]) for i in range(dim)}

def build_matrix(
    rows: list[dict],
    G: nx.DiGraph | None = None,
    tiers: list[int] | None = None,
    category_map: dict[str, int] | None = None,
    vocab: list[str] | None = None,
    embeddings: dict[str, np.ndarray] | None = None,
    embedding_dim: int = EMBEDDING_DIM,
) -> tuple[np.ndarray, np.ndarray, list[str]]:
    """Build feature matrix X and label vector y.

    Args:
        rows: list of market dicts (from load_markets)
        G: ontology DiGraph (needed for tiers 1 and 2)
        tiers: which feature tiers to include (default [0])
        category_map: from build_category_map (tier 0)
        vocab: from build_vocab (tier 0, optional)
        embeddings: from compute_spectral_embeddings (tier 2)
        embedding_dim: dimensionality for tier 2

    Returns:
        (X, y, feature_names) where X is (n, d), y is (n,) in {0, 1}.
    """
    if tiers is None:
        tiers = [0]
    if not rows:
        return np.empty((0, 0)), np.empty(0), []

    predicate_list = _get_predicate_list(G) if G is not None else []

    feat_dicts: list[dict[str, float]] = []
    for row in rows:
        f: dict[str, float] = {}

        if 0 in tiers:
            f.update(extract_tier0(row, category_map, vocab))

        if 1 in tiers:
            if G is not None:
                f.update(extract_tier1(row, G, predicate_list))
            else:
                f.update(_empty_tier1(predicate_list))

        if 2 in tiers:
            if G is not None and embeddings is not None:
                f.update(extract_tier2(row, G, embeddings, embedding_dim))
            else:
                f.update({f"emb_{i}": 0.0 for i in range(embedding_dim)})

        feat_dicts.append(f)

    names = list(feat_dicts[0].keys())
    X = np.array([[fd.get(n, 0.0) for n in names] for fd in feat_dicts], dtype=np.float64)
    y = np.array([r.get("resolved_yes", 0) for r in rows], dtype=np.float64)

    nan_mask = np.isnan(X)
    if nan_mask.any():
        X = np.nan_to_num(X, nan=0.0)

    return X, y, names

def _safe_float(v: Any, default: float = 0.0) -> float:
    try:
        f = float(v)
        return default if (math.isnan(f) or math.isinf(f)) else f
    except (TypeError, ValueError):
        return default


def _parse_json(v: Any, default: Any = None) -> Any:
    if isinstance(v, str):
        try:
            return json.loads(v)
        except (json.JSONDecodeError, ValueError):
            return default
    return v if v is not None else default


def _tokenize(text: str) -> list[str]:
    return re.findall(r"[a-z0-9]+", text.lower())


def _days_to_end(row: dict) -> float:
    end = row.get("end_date", "")
    closed = row.get("closed_time", "")
    if not end:
        return 30.0
    try:
        from datetime import datetime
        fmts = ["%Y-%m-%dT%H:%M:%S.%fZ", "%Y-%m-%dT%H:%M:%SZ", "%Y-%m-%d"]
        end_dt = None
        for fmt in fmts:
            try:
                end_dt = datetime.strptime(end[:26].rstrip("Z") + "Z", fmt)
                break
            except ValueError:
                continue
        if end_dt is None:
            return 30.0
        if closed:
            close_dt = None
            for fmt in fmts:
                try:
                    close_dt = datetime.strptime(closed[:26].rstrip("Z") + "Z", fmt)
                    break
                except ValueError:
                    continue
            if close_dt:
                return max(0.0, (end_dt - close_dt).total_seconds() / 86400)
        return 30.0
    except Exception:
        return 30.0

This repo runs autonomous ML experiments on prediction market forecasting with ontology-derived features.

Setup

Before starting an experiment loop:

  1. Propose a run tag based on today's date, e.g. apr5.
  2. Create a fresh branch: git checkout -b autoresearch/<tag>.
  3. Read README.md, prepare.py, and train.py.
  4. Verify the generated data snapshot exists and is current:
  5. Required files: data/ontology.json, data/resolved.csv, data/snapshot.json
  6. If any are missing, or uv run prepare.py reports Snapshot: stale / Snapshot: invalid, run: uv run python scripts/export_data.py --pred-markets-dir ../prediction-markets
  7. If the exporter reports missing resolved markets upstream, seed prediction-markets first with: uv run python scripts/refresh_markets.py --include-resolved
  8. Initialize results.tsv with the header row shown below if it does not exist.
  9. Run uv run prepare.py once before editing train.py.

Ground truth constraints

  • Only edit train.py.
  • Do not modify prepare.py, scripts/export_data.py, program.md, or data/.
  • Do not change labels, split logic, or metric definitions.
  • Do not add dependencies.

prepare.py is the fixed evaluation harness. It owns:

  • data loading
  • snapshot verification
  • leakage-safe feature extraction
  • baselines
  • metric computation

Data semantics

The exported ontology is filtered before Tier 1 / Tier 2 features are built:

  • Remove nodes whose first_seen is after the last training market's closed_time
  • Remove nodes tagged with resolution or ml_training
  • Remove edges whose timestamp is after the same cutoff

This is required to avoid post-resolution leakage into validation features.

Current snapshot caveat

With the current exported data snapshot, the leakage-safe cutoff removes all ontology edges. Tier 1 and Tier 2 therefore collapse to zero-valued features today. That is the honest result for this dataset. Do not "fix" this by bypassing the temporal filter.

Feature tiers

  • Tier 0: market metadata only
  • Tier 1: graph-structural features from the leakage-safe feature graph
  • Tier 2: spectral embeddings from the same leakage-safe feature graph

What to optimize

Primary objective: minimize validation Brier score.

Secondary objectives:

  • keep the code simple
  • avoid unnecessary VRAM growth
  • preserve the stdout contract

The 5-minute wall-clock budget is shared across all configured seeds.

Expected commands

Use these entrypoints:

  • uv run prepare.py
  • uv run train.py

Use prepare.py to understand the current snapshot:

  • snapshot status
  • leakage audit
  • baseline metrics for tiers [0], [0,1], [0,1,2]

First experiments

Start with these baselines before more creative changes:

  1. uv run prepare.py and record the logistic/tree baselines for each tier set.
  2. uv run train.py with the default config in train.py.
  3. Run a Tier 0 only ablation.
  4. Run a Tier 0+1+2 ablation.

Interpret the results honestly:

  • On the current snapshot, graph tiers are expected to provide no extra signal after filtering.
  • Improvements should therefore come from better use of Tier 0 features unless upstream data changes.

Stdout contract

train.py must always print these lines:

brier: <float>
log_loss: <float>
calibration_error: <float>
auc: <float>
training_seconds: <float>
peak_vram_mb: <float>
tiers: [..]
seeds: [..]

It may also print these lines when snapshot provenance is available:

snapshot_hash: <hex>
graph_content_hash: <hex>

Other informational lines are allowed, but keep the brier: line singular and grep-friendly.

Logging results

Log every experiment to results.tsv using tab-separated values.

Header:

commit  brier   brier_std   log_loss    calibration_error   auc memory_gb   tiers   seeds   status  snapshot    description

Columns:

  1. short git commit hash
  2. mean Brier score
  3. Brier std across seeds
  4. mean log-loss
  5. mean calibration error
  6. mean AUC
  7. peak VRAM in GB (peak_vram_mb / 1024, rounded to one decimal)
  8. tiers, e.g. 0 or 0,1,2
  9. seeds, e.g. 42,123,456
  10. status: keep, discard, or crash
  11. snapshot hash prefix (first 8 chars), or blank if unavailable
  12. short experiment description

results.tsv is runtime output and should stay untracked.

Experiment loop

Repeat:

  1. Inspect the current branch and commit.
  2. Make one coherent change in train.py.
  3. Commit the change.
  4. Run uv run train.py > run.log 2>&1.
  5. Extract metrics with: grep "^brier:\|^brier_std:\|^log_loss:\|^calibration_error:\|^auc:\|^peak_vram_mb:\|^tiers:\|^seeds:\|^snapshot_hash:" run.log
  6. If the metrics are missing, inspect the crash with tail -n 50 run.log.
  7. Append a row to results.tsv.
  8. Keep the commit only if the result is meaningfully better.
  9. Otherwise revert to the previous experiment commit.

Timeout rule:

  • If a run takes more than 10 minutes, kill it and record it as a crash.

Keep / discard heuristic

  • Keep clearly better Brier scores.
  • Prefer simpler changes when scores are effectively tied.
  • Do not keep improvements that depend on leaking post-resolution graph data.

#!/usr/bin/env python3
"""Export the reproducible onto-research data snapshot from pred-markets.

Copies ``data/ontology.json``, exports the ``resolved_markets`` SQLite table
to ``data/resolved.csv``, and writes ``data/snapshot.json`` with file hashes
and graph metadata. This gives onto-research a flat-file, reproducible input
bundle with no runtime dependency on prediction-markets.

Usage:
    python scripts/export_data.py
    python scripts/export_data.py --pred-markets-dir /path/to/prediction-markets
    python scripts/export_data.py --db-path /path/to/memory.db --ontology-path /path/to/ontology.json
"""
from __future__ import annotations

import argparse
import csv
import hashlib
import json
import shutil
import sqlite3
from datetime import datetime, timezone
from pathlib import Path

_HERE = Path(__file__).resolve().parent
_PROJECT = _HERE.parent
_DATA_DIR = _PROJECT / "data"

_DEFAULT_PRED_MARKETS = _PROJECT.parent / "prediction-markets"

_RESOLVED_COLS = [
    "market_id",
    "question",
    "category",
    "tags",
    "volume",
    "liquidity",
    "implied_prob_at_close",
    "resolved_yes",
    "end_date",
    "closed_time",
    "fetched_at",
]


def _existing_resolved_rows(path: Path) -> int:
    """Return the number of existing resolved rows in a CSV, else 0."""
    if not path.exists():
        return 0
    with open(path, encoding="utf-8") as f:
        reader = csv.DictReader(f)
        return sum(1 for _ in reader)


def _export_ontology(src: Path, dst: Path) -> int:
    """Copy ontology JSON and return triple count."""
    if not src.exists():
        print(f"  [SKIP] ontology.json not found at {src}")
        return 0
    dst.parent.mkdir(parents=True, exist_ok=True)
    shutil.copy2(src, dst)
    data = json.loads(src.read_text(encoding="utf-8"))
    n_nodes = len(data.get("nodes", []))
    n_edges = len(data.get("links", []))
    print(f"  Copied ontology.json: {n_nodes} nodes, {n_edges} edges")
    return n_edges


def _export_resolved(db_path: Path, dst: Path) -> tuple[str, int]:
    """Export resolved_markets to CSV, or reuse an existing CSV if needed.

    Returns ``(status, row_count)`` where status is one of:
    ``exported``, ``reused``, or ``missing``.
    """
    if not db_path.exists():
        print(f"  [SKIP] database not found at {db_path}")
        existing_rows = _existing_resolved_rows(dst)
        if existing_rows > 0:
            print(f"  [WARN] reusing existing resolved.csv: {existing_rows} markets")
            return "reused", existing_rows
        return "missing", 0
    dst.parent.mkdir(parents=True, exist_ok=True)

    con = sqlite3.connect(str(db_path))
    con.row_factory = sqlite3.Row
    try:
        cursor = con.execute(
            "SELECT * FROM resolved_markets ORDER BY closed_time ASC"
        )
        rows = cursor.fetchall()
    except sqlite3.OperationalError as exc:
        print(f"  [SKIP] resolved_markets table: {exc}")
        existing_rows = _existing_resolved_rows(dst)
        if existing_rows > 0:
            print(f"  [WARN] reusing existing resolved.csv: {existing_rows} markets")
            return "reused", existing_rows
        return "missing", 0
    finally:
        con.close()

    if not rows:
        print("  [SKIP] resolved_markets table is empty")
        existing_rows = _existing_resolved_rows(dst)
        if existing_rows > 0:
            print(f"  [WARN] reusing existing resolved.csv: {existing_rows} markets")
            return "reused", existing_rows
        return "missing", 0

    with open(dst, "w", newline="", encoding="utf-8") as f:
        writer = csv.DictWriter(f, fieldnames=_RESOLVED_COLS)
        writer.writeheader()
        for row in rows:
            d = dict(row)
            writer.writerow({c: d.get(c, "") for c in _RESOLVED_COLS})

    yes = sum(1 for r in rows if dict(r).get("resolved_yes"))
    dates = [dict(r).get("closed_time", "") for r in rows if dict(r).get("closed_time")]
    date_range = f"{min(dates)} .. {max(dates)}" if dates else "N/A"
    print(f"  Exported {len(rows)} resolved markets to CSV")
    print(f"  YES rate: {yes}/{len(rows)} ({100 * yes / len(rows):.1f}%)")
    print(f"  Date range: {date_range}")
    return "exported", len(rows)


def _sha256_file(path: Path) -> str:
    """Compute SHA-256 hex digest of a file."""
    h = hashlib.sha256()
    with open(path, "rb") as f:
        for chunk in iter(lambda: f.read(1 << 16), b""):
            h.update(chunk)
    return h.hexdigest()


def _write_snapshot(
    ont_dst: Path, csv_dst: Path, n_rows: int,
) -> str:
    """Write data/snapshot.json with file hashes and graph stats."""
    csv_hash = _sha256_file(csv_dst) if csv_dst.exists() else ""
    ont_hash = _sha256_file(ont_dst) if ont_dst.exists() else ""

    node_count = 0
    edge_count = 0
    if ont_dst.exists():
        data = json.loads(ont_dst.read_text(encoding="utf-8"))
        node_count = len(data.get("nodes", []))
        edge_count = len(data.get("links", []))

    # Kernel-level content hash (if ontokernel is available)
    graph_content_hash = ""
    try:
        from ontokernel import OntologyGraph
        from ontokernel.config import KernelConfig
        config = KernelConfig(
            backend="networkx",
            persist_path=str(ont_dst),
            default_namespace="polymarket",
        )
        g = OntologyGraph(config=config)
        g.load()
        snap_obj = g.snapshot()
        graph_content_hash = snap_obj.content_hash
    except Exception:
        pass

    composite = hashlib.sha256((csv_hash + ont_hash).encode()).hexdigest()

    snapshot = {
        "resolved_csv_sha256": csv_hash,
        "ontology_json_sha256": ont_hash,
        "graph_content_hash": graph_content_hash,
        "export_timestamp": datetime.now(timezone.utc).isoformat(),
        "row_count": n_rows,
        "node_count": node_count,
        "edge_count": edge_count,
        "snapshot_hash": composite,
    }

    snap_path = _DATA_DIR / "snapshot.json"
    snap_path.write_text(json.dumps(snapshot, indent=2), encoding="utf-8")
    print(f"\n  Snapshot written to {snap_path}")
    print(f"    snapshot_hash: {composite[:16]}...")
    if graph_content_hash:
        print(f"    graph_content_hash: {graph_content_hash[:16]}...")
    return composite


def main() -> None:
    parser = argparse.ArgumentParser(
        description="Export data from pred-markets for onto-research experiments"
    )
    parser.add_argument(
        "--pred-markets-dir",
        type=Path,
        default=_DEFAULT_PRED_MARKETS,
        help="Root of the prediction-markets repo (default: ../prediction-markets)",
    )
    parser.add_argument(
        "--db-path",
        type=Path,
        default=None,
        help="Direct path to memory.db (overrides --pred-markets-dir)",
    )
    parser.add_argument(
        "--ontology-path",
        type=Path,
        default=None,
        help="Direct path to ontology.json (overrides --pred-markets-dir)",
    )
    args = parser.parse_args()

    ont_src = args.ontology_path or (args.pred_markets_dir / "data" / "ontology.json")
    db_src = args.db_path or (args.pred_markets_dir / "data" / "memory.db")

    print(f"Exporting from pred-markets ({args.pred_markets_dir})")
    print(f"  ontology: {ont_src}")
    print(f"  database: {db_src}")
    print()

    ont_dst = _DATA_DIR / "ontology.json"
    csv_dst = _DATA_DIR / "resolved.csv"

    n_edges = _export_ontology(ont_src, ont_dst)
    resolved_status, n_rows = _export_resolved(db_src, csv_dst)

    # Write snapshot.json with file hashes + graph content hash
    snapshot_hash = None
    if n_edges > 0 and n_rows > 0:
        snapshot_hash = _write_snapshot(ont_dst, csv_dst, n_rows)

    print()
    if n_edges == 0 and n_rows == 0:
        print("No data exported. Make sure pred-markets has been seeded:")
        print("  cd ../prediction-markets")
        print("  uv run python scripts/refresh_markets.py --include-resolved")
    else:
        print(f"Done. Data written to {_DATA_DIR}/")
        print(f"  ontology.json  ({n_edges} edges)")
        if resolved_status == "exported":
            print(f"  resolved.csv   ({n_rows} markets, exported)")
        elif resolved_status == "reused":
            print(f"  resolved.csv   ({n_rows} markets, reused existing file)")
        else:
            print("  resolved.csv   (missing)")
        if snapshot_hash:
            print(f"  snapshot.json  ({snapshot_hash[:16]}...)")


if __name__ == "__main__":
    main()

"""Editable training surface for ontology-ML experiments.

THIS IS THE ONLY FILE THE AUTORESEARCH AGENT MAY EDIT.

Contract:
  - Must print exactly one line matching ``brier: <float>`` to stdout.
  - Must print ``training_seconds: <float>`` and ``peak_vram_mb: <float>``.
  - Must print ``log_loss: <float>``, ``calibration_error: <float>``, ``auc: <float>``.
  - May additionally print ``snapshot_hash: <hex>`` and ``graph_content_hash: <hex>``
    when snapshot provenance is available.
  - Must complete within TIME_BUDGET (default 300 s / 5 min).
  - ``train()`` returns ``(model, brier)``.

The agent can freely edit: CONFIG values, model architecture, optimizer,
training loop, feature tier selection, batch size, etc.
"""
from __future__ import annotations

import sys
import time
from pathlib import Path

import numpy as np
import torch
import torch.nn as nn

from prepare import (
    build_feature_graph,
    build_category_map,
    build_matrix,
    build_vocab,
    compute_spectral_embeddings,
    evaluate_all,
    evaluate_brier,
    load_markets,
    load_ontology_graph,
    load_ontology_kernel,
    time_split,
    verify_snapshot,
)

# ========================= CONFIG (agent edits this) =========================
DEPTH = 2                          # number of hidden layers
HIDDEN = 64                        # size of each hidden layer
DROPOUT = 0.1                      # dropout rate (0 to disable)
LR = 0.001                         # learning rate
WEIGHT_DECAY = 1e-4                # AdamW weight decay
BATCH_SIZE = 256                   # mini-batch size
MAX_EPOCHS = 200                   # max training epochs
TIME_BUDGET = 300                  # 5 minutes wall-clock (shared across all seeds)
FEATURE_TIERS = [0, 1]            # which tiers to include: 0=tabular, 1=graph, 2=embeddings
USE_VOCAB = False                  # bag-of-words features from question text
MAX_VOCAB = 200                    # vocab size if USE_VOCAB enabled
SEEDS = [42, 123, 456]            # multi-seed runs for reproducibility
MAX_VRAM_MB = 6500                 # 8GB RTX 3050 minus 1.5GB headroom
USE_AMP = False                    # mixed precision (disabled by default)
EXCLUDE_SOURCES = {"resolution", "ml_training"}
# =============================================================================


class TinyForecaster(nn.Module):
    """Small MLP — the agent can freely swap architecture."""

    def __init__(self, n_features: int):
        super().__init__()
        layers: list[nn.Module] = []
        prev = n_features
        for _ in range(DEPTH):
            layers.append(nn.Linear(prev, HIDDEN))
            layers.append(nn.ReLU())
            if DROPOUT > 0:
                layers.append(nn.Dropout(DROPOUT))
            prev = HIDDEN
        layers.append(nn.Linear(prev, 1))
        layers.append(nn.Sigmoid())
        self.net = nn.Sequential(*layers)

    def forward(self, x: torch.Tensor) -> torch.Tensor:
        return self.net(x).squeeze(-1)


def _train_single_seed(
    seed: int,
    device: torch.device,
    X_t: torch.Tensor,
    y_t: torch.Tensor,
    X_v: torch.Tensor,
    y_val: np.ndarray,
    n_features: int,
    per_seed_budget: float,
) -> tuple[nn.Module, dict[str, float]]:
    """Train one seed — deterministic init, returns (model, metrics_dict)."""
    torch.manual_seed(seed)
    np.random.seed(seed)
    if device.type == "cuda":
        torch.cuda.manual_seed_all(seed)

    model = TinyForecaster(n_features).to(device)
    optimizer = torch.optim.AdamW(model.parameters(), lr=LR, weight_decay=WEIGHT_DECAY)
    criterion = nn.BCELoss()

    use_amp = USE_AMP and device.type == "cuda"
    scaler = torch.amp.GradScaler("cuda") if use_amp else None

    n = len(X_t)
    best_brier = 1.0
    start = time.time()

    for epoch in range(MAX_EPOCHS):
        if time.time() - start > per_seed_budget:
            break

        model.train()
        perm = torch.randperm(n, device=device)

        for i in range(0, n, BATCH_SIZE):
            idx = perm[i : i + BATCH_SIZE]
            xb, yb = X_t[idx], y_t[idx]

            optimizer.zero_grad()
            if use_amp and scaler is not None:
                with torch.amp.autocast("cuda"):
                    pred = model(xb)
                    loss = criterion(pred, yb)
                scaler.scale(loss).backward()
                scaler.step(optimizer)
                scaler.update()
            else:
                pred = model(xb)
                loss = criterion(pred, yb)
                loss.backward()
                optimizer.step()

        model.eval()
        with torch.no_grad():
            val_probs = model(X_v).cpu().numpy()
        brier = evaluate_brier(y_val, val_probs)
        best_brier = min(best_brier, brier)

    model.eval()
    with torch.no_grad():
        final_probs = model(X_v).cpu().numpy()
    metrics = evaluate_all(y_val, final_probs)
    metrics["best_epoch_brier"] = best_brier

    return model, metrics


def train(csv_path: str | None = None) -> tuple[nn.Module, float]:
    """Full training pipeline — multi-seed, returns (best_model, mean_brier)."""
    device = torch.device("cuda" if torch.cuda.is_available() else "cpu")
    start = time.time()

    if device.type == "cuda":
        torch.cuda.reset_peak_memory_stats()

    # Load data once
    G = load_ontology_graph()
    kernel = load_ontology_kernel()
    markets = load_markets() if csv_path is None else load_markets(Path(csv_path))
    train_rows, val_rows = time_split(markets)

    # Snapshot provenance
    snap_hash = verify_snapshot()

    # Compute leakage-safe cutoff from last training row
    cutoff_ts: float | None = None
    if train_rows:
        last_closed = train_rows[-1].get("closed_time", "")
        if last_closed:
            try:
                from datetime import datetime
                cutoff_dt = datetime.fromisoformat(last_closed.replace("Z", "+00:00"))
                cutoff_ts = cutoff_dt.timestamp()
            except (ValueError, TypeError):
                pass

    feature_graph = build_feature_graph(
        G, cutoff_timestamp=cutoff_ts, exclude_sources=EXCLUDE_SOURCES
    )

    # Kernel graph snapshot (content fingerprint)
    graph_content_hash: str | None = None
    if kernel is not None:
        try:
            snap_obj = kernel.snapshot()
            graph_content_hash = snap_obj.content_hash
        except Exception:
            pass

    if len(train_rows) < 10 or len(val_rows) < 5:
        print("brier: 0.250000", flush=True)
        print("log_loss: 0.693147", flush=True)
        print("calibration_error: 0.000000", flush=True)
        print("auc: 0.500000", flush=True)
        print("training_seconds: 0.0", flush=True)
        print("peak_vram_mb: 0.0", flush=True)
        if snap_hash:
            print(f"snapshot_hash: {snap_hash}", flush=True)
        if graph_content_hash:
            print(f"graph_content_hash: {graph_content_hash}", flush=True)
        return nn.Linear(1, 1), 0.25

    cat_map = build_category_map(train_rows)
    vocab = build_vocab(train_rows, MAX_VOCAB) if USE_VOCAB else None

    embeddings = None
    if 2 in FEATURE_TIERS and feature_graph.number_of_nodes() > 0:
        embeddings = compute_spectral_embeddings(feature_graph)

    X_train, y_train, feat_names = build_matrix(
        train_rows, feature_graph, tiers=FEATURE_TIERS,
        category_map=cat_map, vocab=vocab, embeddings=embeddings,
    )
    X_val, y_val, _ = build_matrix(
        val_rows, feature_graph, tiers=FEATURE_TIERS,
        category_map=cat_map, vocab=vocab, embeddings=embeddings,
    )

    n_features = X_train.shape[1]

    X_t = torch.tensor(X_train, dtype=torch.float32, device=device)
    y_t = torch.tensor(y_train, dtype=torch.float32, device=device)
    X_v = torch.tensor(X_val, dtype=torch.float32, device=device)

    # Trivial baselines (zero-cost reference)
    train_mean = float(np.mean(y_train))
    baseline_brier = evaluate_brier(y_val, np.full(len(y_val), train_mean))
    market_preds = np.array(
        [float(r.get("implied_prob_at_close", 0.5)) for r in val_rows], dtype=np.float64
    )
    market_brier = evaluate_brier(y_val, market_preds)

    # Multi-seed training
    per_seed_budget = TIME_BUDGET / len(SEEDS)
    all_metrics: list[dict[str, float]] = []
    best_model: nn.Module = nn.Linear(1, 1)
    best_brier = 1.0

    for seed in SEEDS:
        model, metrics = _train_single_seed(
            seed, device, X_t, y_t, X_v, y_val,
            n_features, per_seed_budget,
        )
        all_metrics.append(metrics)
        if metrics["brier"] < best_brier:
            best_brier = metrics["brier"]
            best_model = model

    elapsed = time.time() - start
    peak_vram = torch.cuda.max_memory_allocated() / 1e6 if torch.cuda.is_available() else 0.0

    # Aggregate metrics across seeds
    metric_keys = ["brier", "log_loss", "calibration_error", "auc"]
    means = {k: float(np.mean([m[k] for m in all_metrics])) for k in metric_keys}
    stds = {k: float(np.std([m[k] for m in all_metrics])) for k in metric_keys}

    # Attach metadata for artifact storage compatibility
    best_model._onto_research_feature_names = feat_names  # type: ignore[attr-defined]
    best_model._onto_research_category_map = cat_map  # type: ignore[attr-defined]
    best_model._onto_research_vocab = vocab or []  # type: ignore[attr-defined]
    best_model._onto_research_tiers = FEATURE_TIERS  # type: ignore[attr-defined]

    # Output contract (backward-compatible: brier: on its own line)
    print(f"brier: {means['brier']:.6f}", flush=True)
    print(f"log_loss: {means['log_loss']:.6f}", flush=True)
    print(f"calibration_error: {means['calibration_error']:.6f}", flush=True)
    print(f"auc: {means['auc']:.6f}", flush=True)
